# Trader Behavior vs Market Sentiment Analysis

## Objective
Analyze how Bitcoin market sentiment influences trader profitability, trading activity, win rates, and position sizing.

## Datasets
1. Bitcoin Fear & Greed Index
2. Hyperliquid Historical Trading Data

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Key Findings
- Highest average profits occurred during Greed periods.
- Fear periods generated the highest trading activity.
- Extreme Greed showed the highest win rate.
- Traders increased position sizes during extreme sentiment conditions.
